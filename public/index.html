How to Use

Text a bus stop code to get arrival times:

  • 308209 - All buses at this stop
  • 308209 B63 - Only B63 buses

Finding Your Stop Code

  • Visit bustime.mta.info
  • Search for your stop or route
  • Look for the 6-digit code (e.g., "308209")
  • Or check the sign at your bus stop

Features

  • Real-time GPS tracking
  • Shows stops away
  • Filter by route
  • Works for all NYC buses
